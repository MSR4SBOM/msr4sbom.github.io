MSR4SBOM

Mining Software Repositories for Enhanced Software Bills of Materials

MSR4SBOM is a project that aims to deliver a framework that analyzes the content of software repositories and Software Bill of Materials (SBOMs) to provide context-sensitive recommendations.

Find Out More

PROJECT

MSR4SBOM will support software engineers with:

Dependencies

Management of Software Supply Chains

Provenance

Tracking the Origin of Components and Code

Licensing

Detection of Licensing Issues

Security

Recommendations for Vulnerable Dependencies
